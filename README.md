# Sistema-de-Loja
Sistema de Gerenciamento de Estoque feito por Arthur Tabbi e Gabriela oliveira.

Descrição do Projeto:

Estamos no terceiro ano de ensino médio e Este projeto consiste no desenvolvimento de um sistema de gerenciamento de estoque para uma loja, envolvendo modelagem de banco de dados e implementação em Java. O sistema tem como finalidade organizar, controlar e registrar as movimentações de produtos, garantindo maior precisão e eficiência na gestão do estoque.

Justificativa:

O tema foi escolhido aceito pelos dois membros do grupo. O tema foi escolhido pela necessidade e a relevância do controle de estoque no contexto comercial e a oportunidade de aplicar, de forma prática, os conhecimentos adquiridos em banco de dados e programação orientada a objetos. Além disso, trata-se de um projeto viável tecnicamente dentro do nível de formação atual.

Funcionalidades Principais:

Edição e remoção de produtos
Controle de quantidade em estoque

O sistema deverá receber:

Dados do produto (nome, preço, quantidade)
Custo ( venda do produto e reposição do produto)
