# Sistema-de-Loja
Sistema de Gerenciamento de Estoque

Descrição do Projeto:

Este projeto consiste no desenvolvimento de um sistema de gerenciamento de estoque para uma loja, envolvendo modelagem de banco de dados e implementação em Java. O sistema tem como finalidade organizar, controlar e registrar as movimentações de produtos, garantindo maior precisão e eficiência na gestão do estoque.

Justificativa:

O tema foi escolhido por dois estudantes do terceiro ano do curso técnico em Tecnologia da Informação, considerando a relevância do controle de estoque no contexto comercial e a oportunidade de aplicar, de forma prática, os conhecimentos adquiridos em banco de dados e programação orientada a objetos. Além disso, trata-se de um projeto viável tecnicamente dentro do nível de formação atual.

Funcionalidades Principais:

Cadastro, edição e remoção de produtos
Controle de quantidade em estoque
Registro de entradas e saídas de mercadorias
Registro de vendas
Previsão de Entradas do Sistema

O sistema deverá receber:

Dados do produto (nome, código, categoria, preço de custo, preço de venda, quantidade)
Informações de movimentação (tipo, produto, quantidade, data)
Dados de venda (produtos, quantidades e valor total)
